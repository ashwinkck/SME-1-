Each of those corresponds directly to something in your code.

langchain
Core framework.

langchain-community
Contains loaders like PyPDFLoader and vector stores like FAISS.

langchain-text-splitters
Contains RecursiveCharacterTextSplitter.

langchain-huggingface
Provides the HuggingFaceEmbeddings wrapper.

sentence-transformers
This is the actual embedding model engine used by "all-MiniLM-L6-v2".

faiss-cpu
Vector database for similarity search.

groq
Client for calling Groq-hosted LLMs.

pypdf
Used internally by PyPDFLoader to read PDFs.
